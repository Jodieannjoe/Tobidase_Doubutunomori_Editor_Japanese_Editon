どうぶつの森セーブエディダー

どうぶつの森セーブエディダーどうぶつの森セーブエディダーはセーブデータを編集できます。
機能:
  • ACNLセーブデータ（Welcome Amiiboを含む）を編集できます
  • あなたの町を編集することができます
    • 土地、川、滝、池
    • 名前、市庁舎、駅の屋根の色
    • 建物、家、岩などを自分で移動させれる
  • プレイヤーキャラクター（名前、顔、性別、TPC写真、在庫、部屋）を編集できます
  • 村人（動物、キャンプ場、キャラバンゾーン）を編集できます
  • その他のクールなもの
    • あなたの町に完璧な果樹をすべて置いてください
    • 両方の警察署をあなたの町に置いてください
    • ビーチ、川、島に何でも入れてください
    • さまざまな広場の木をどこにでも置く
    • NPCをあなたの町に参加させましょう
    • 冬でも日焼けする
    • 地面の草の形を変える
    • 未使用のプレーヤーのパターンを地面に置きます
    • ...その他！

AC：NLセーブデータをダンプして注入する方法

要件

  • 改造したニンテンドー3DS/XL,Newニンテンドー3DS/XL,ニンテンドー2DS,またはNewニンテンドー2DS XL
  • Welcome amiiboアップデートの有無にかかわらずAC：NLの小売/デジタルバージョン
  • 更新されたWebブラウザ (推奨: Firefox)

3DSを改造する方法

ガイド ではあなたの3DSを改造するための事が書かれています。

セーブデータを抽出する方法

  1. 3DSを開く
  2. CheckPointを開きます
  3. AC：NLアイコンを選択し、Aを押します
  4. バックアップ（L）を選択し、セーブゲームのバックアップの名前を入力します
  5. コンソールの電源を切り、SDカードを取り出して、PCに挿入します

セーブデータを挿入

  1. SDカードを参照し、/3DS/Checkpoint/saves/AnimalCrossing/[自分のセーブデータの名前]/ フォルダ全体のバックアップを作成します
  2. AC：NLで/3DS/Checkpoint/saves/Animal Crossing/[自分のセーブデータの名前]/garden.dat（またはgarden_plus.dat）をエディタで編集して、自分の好みの町にする、したら保存をする。(その後、編集したデータがダウンロードされます。)
  3. 編集したデータをSD に/3DS/Checkpoint/saves/ Animal Crossing/[自分のセーブデータの名前]/garden.dat（またはgarden_plus.dat）に上書きする
  4. 3DSにSDカードを挿入します
  5. 3DSを起動する。
  6. CheckPointを開きます
  7. AC：NLアイコンを選択し、Aを押します
  8. 復元（R）を選択し、[自分のセーブデータの名前]を選択します
  9. AC：NLを実行してゲーム内に入ると、変更が反映されたら成功です。

FAQ

編集したセーブデータでオンラインをプレイした場合、BANされますか？

BANされる可能性があります。TPC写真を利用すると可能性が高まります。

部屋の家具を回転させるにはどうすればよいですか？

目的の家具を右クリックや左クリックして回転させれます。

完璧な果樹を配置するにはどうすればよいですか？(日本語が登録されていません。)

[現在のアイテム]ドロップダウンメニューで目的のツリーを選択し、[フラグ1]ドロップダウンメニューで[パーフェクト4]を選択します。 非在来種の完璧な果樹を置くこともできます！
珍しいミックスパーフェクトフルーツツリーを配置することもできます（2つの通常のフルーツ+1パーフェクトフルーツ）。 町にある通常の果樹のいずれかを右クリックし、[フラグ2]ドロップダウンメニューで[0x01ミックスパーフェクト]を選択して、マップ（または配置する場所）で上書きします。

NPCを注入しましたが、町に来ません。

NPCはどちらもRVロックされているため、村人として合法的にあなたの町に来ることはできません。
NPCを村人として持つ唯一の方法は、NPCをあなたの町の既存の村人に直接注入することです。

あなたの編集者は私のセーブデータをグリッチしました！

いいえ、それは私の編集者ではなく、あなたです。
エディターはクールなことをすることができますが、それは危険なツールでもあり、それを使用している間はあなただけが責任を負います。 セーブデータのバックアップを常に保管してください。
バグを見つけたと思われる場合は、フィードバックをここに投稿してください。

ワイルドワールドのような建物の種を作成できますか？

いいえ。

セーブデータに不具合が発生しないようにするにはどうすればよいですか？

  • 少なくとも池が必要です
  • 2つの滝（壁と海）が必要です。
  • 町の広場の木が必要です
  • 2つの傾斜が必要です：木を成長させる機能があるのでそこで成長させてください。
  • 木を成長させる機能で奇妙なことをしないでください：有効な木の構造を維持するようにしてください
  • 少なくとも2つの岩を保持します：必要に応じてマップエディタを使用してそれらを移動します（右クリックして岩を「コピー」し、任意の場所に配置してから元の岩を削除します）
  • 町の広場の木に建物のための十分な空きスペースを確保する：特別な訪問者（Redd、Gracie、Katrina ...）がテントをそこに置こうとし、十分なスペースがない場合、ゲームはフリーズします
  • 建物を移動するときは注意してください：エディタを使用して建物の配置を調整できますが、家を水中に配置するなどの行為はしないでください

ゲームは私のセーブデータが壊れていると言っています。セーブゲームを復元できますか？

つまり、古いセーブゲームを挿入し、ゲームがアンチチート保護システムを使用してそれをブロックしたことを意味します。

心配しないでください、あなたはデータを復元することができます。安全なNAND値を更新する必要があります。
  1. 復元する古いセーブデータのバックアップを作成します
  2. ゲームを再開して保存します。この新しいAC：NLセーブゲームのダンプを作成します。
  3. エディターで古い garden.dat / garden_plus.datセーブデータを開きます
  4. [その他]タブに移動し、[セキュリティで保護された値]の横にある鉛筆をクリックして、最新のgarden.dat / garden_plus.datをここに読み込みます。
    これにより、古いセーブゲームがコンソールの有効なセーブデータに変わります。

RAMハッキング方法はどうなりましたか？

WelcomeamiiboアップデートのRAMメソッドのサポートは終了しました。そして、二度と戻ってこない、ごめんなさい。
その理由は、RAM方式が一部の古いコンソールでしか利用できなかったためです。 amiiboのアップデートでは、コンソールファームウェアが11.xである必要があります。これにより、AC：NL NAMハッキング方法を可能にするブラウザエクスプロイトが自動的に強制終了されます。

基本的に、任天堂は意図せずにRAMハッキング方法をなくしました。
